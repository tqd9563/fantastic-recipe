# 首页（Dashboard）

**核心设计理念**：聚焦当下（今天吃什么），兼顾动力（热力图）与未来引导（快捷入口）。

## 1. 顶部：个人状态栏

- 左侧：日期，当前时间（精确到时分秒）
- 右侧：做饭热力图（mini版）：

  - 风格：Github风格的热力图，直观地看到最近做饭的勤快程度（默认显示时间近2个月）
  - 文案：可配一句动态文案：最近2个月已经做饭{X}天


## 2. 核心区域：今日菜单

- **有计划时**

  - **菜单概览条**

    - 位置：大卡片上方
    - 形式：简洁的文本列表
    - 内容：

      - 标题：“今日菜单：红烧排骨、清炒茼蒿“
      - 视觉：字体适中


  - **菜品详情卡片**

    - **展示形式**：横向大卡片轮播（Carousel）
    - **卡片内容**：

      - **左侧大图**：诱人的成品图。
      - **右侧信息**：

        - **菜名**：如“红烧排骨”。
        - **标签**：评分⭐️⭐️⭐️


    - **卡片交互**：

      - 点击卡片，跳转至食谱库中对应食谱的详情页面。



- **无计划时**

  - **展示形式**：一个带有微插画的空白状态卡片。
  - **文案**：“今天还没想好吃什么”


## 3. 底部区域：快捷行动舱 (Quick Actions)

位于今日菜单下方，采用 2x2 或 1x3 的网格布局，作为通往其他复杂功能的快速通道。

- **[📅 未来计划] 卡片**

  - **内容**：

    - 如果未来几天有安排：可简略显示明后两天的菜名（如“明天：青椒肉丝...”）。
    - 如果未来几天没有安排：显示”未来暂无计划“

  - **点击**：跳转至 [计划] 页。

- **[🔭 探索灵感] 卡片**

  - **内容**：展示一张精美的随机美食图（来自 Discovery 推荐）。
  - **文案**：“发现新美味”。
  - **点击**：跳转至 [发现] 页。
